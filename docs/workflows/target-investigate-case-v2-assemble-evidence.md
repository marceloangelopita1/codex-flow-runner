# Assemble Evidence

Objetivo: decidir quais evidencias sao necessarias, como coleta-las e como indexa-las auditavelmente.

Saida minima:
- `evidence-index.json`
- `case-bundle.json`

Regras:
- use apenas superficies e estrategias declaradas no manifesto;
- `case-bundle.json` deve referenciar `evidence-index.json`;
- evite duplicacao indiscriminada do inventario bruto.
