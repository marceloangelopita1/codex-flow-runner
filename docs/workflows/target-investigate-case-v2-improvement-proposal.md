# Improvement Proposal

Objetivo: materializar a menor mudança plausível necessária depois de um diagnóstico suficientemente estável.

Saída mínima:
- `improvement-proposal.json`

Regras:
- execute este estágio apenas depois de `diagnosis` com confiança suficiente para delimitar a mudança;
- o foco é propor a correção, não reabrir a análise inteira;
- se o target ainda estiver em migração, a ponte legada pode coexistir, mas a autoridade semântica continua neste estágio v2.
